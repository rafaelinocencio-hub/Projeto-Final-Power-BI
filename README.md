# Projeto Final - Power BI
Este projeto apresenta um dashboard interativo criado no Power BI para análise de desempenho de vendas.

## Indicadores Principais
- Lucro Total por País
- Top 5 Produtos Mais Lucrativos
- Evolução de Lucro e Vendas ao Longo do Tempo
- Distribuição das Vendas por Segmento
